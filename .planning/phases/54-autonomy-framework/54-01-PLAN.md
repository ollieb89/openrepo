# Plan 54-01: Core Interfaces and State Machine

## Goal

Create the `autonomy/` package with foundational interfaces and the 4-state autonomy state machine.

## Context

From 54-CONTEXT.md:
- **4-state model**: planning → executing → complete|escalating
- **Escalating is a state** that pauses work until L2 reviews
- **Self-recovery allowed**: 1 automatic retry attempt from 'blocked' before escalating
- **Distributed ownership**: Each L3 self-reports state via sentinel file or state.json
