# Phase 03 Plan Index: Decision Summaries with Provenance

This phase is split into three executable plans to deliver trustworthy thread summaries with local-first processing and verifiable provenance:

- `03-01-PLAN.md` (Wave 1): storage foundation, decision data models, and privacy-scrubbed thread persistence.
- `03-02-PLAN.md` (Wave 2): local LLM integration (Ollama/Phi-3) and "Convergent Evolution" extraction engine.
- `03-03-PLAN.md` (Wave 3): decision log feed UI, citation/provenance surfaces, and correction/hiding loops.

## Wave Structure

- Wave 1: `03-01-PLAN.md`
- Wave 2: `03-02-PLAN.md` (depends on Wave 1 storage)
- Wave 3: `03-03-PLAN.md` (depends on Wave 2 extraction engine)

- `SUMM-02`: `03-02-PLAN.md`, `03-03-PLAN.md`
