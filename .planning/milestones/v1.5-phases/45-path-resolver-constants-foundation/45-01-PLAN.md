  <action>
Add `from pathlib import Path` import at top of config.py.

Add three functions after the existing constants:

1. `_find_project_root() -> Path` (internal helper):
   - Check `OPENCLAW_ROOT` env var first → return `Path(env_root)` if set
   - Fall back to `Path.home() / ".openclaw"`
   - NEVER use `Path(__file__).parent` — that resolves to install location, not project root
   - Add docstring explaining resolution order

2. `get_project_root() -> Path` (public API):
   - Thin wrapper around `_find_project_root()` — exists as the public export name
   - monitor.py needs this for projects directory enumeration

3. `get_state_path(project_id: str) -> Path`:
   - `project_id` is REQUIRED — no `Optional[str] = None` default, no active-project fallback
   - Check `OPENCLAW_STATE_FILE` env var first → return `Path(env_state)` if set (aligns with container entrypoint.sh)
   - Otherwise derive: `_find_project_root() / "workspace" / ".openclaw" / project_id / "workspace-state.json"`
   - Compute-only — no existence validation (callers validate project upstream)

4. `get_snapshot_dir(project_id: str) -> Path`:
   - `project_id` is REQUIRED — same as get_state_path
   - Derive: `_find_project_root() / "workspace" / ".openclaw" / project_id / "snapshots"`
   - Does NOT check OPENCLAW_STATE_FILE — snapshot dir has no equivalent env var
