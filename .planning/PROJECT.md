# Project: OpenClaw (Grand Architect Protocol)

## Context
OpenClaw is an AI Swarm Orchestration system transitioning to a 3-tier hierarchical architecture (Grand Architect Protocol). This phase focuses on the "PumplAI Pilot" implementation and the "occc" Next.js dashboard for real-time monitoring.

## Objective
Establish the core orchestration layer, physical isolation substrate, and monitoring uplink for the Grand Architect Protocol.

## Tech Stack
- **Core:** OpenClaw CLI, Bun, Docker
- **Frontend:** Next.js 16 (App Router), Tailwind 4, SWR
- **AI:** Gemini 2.5 Flash / 3.0 Pro
- **OS:** Ubuntu 24.04 LTS
