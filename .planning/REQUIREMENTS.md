# Requirements: OpenClaw v1.6 (Agent Autonomy)

**Defined:** 2026-02-25
**Core Value:** Hierarchical AI orchestration with physical isolation — enabling autonomous, secure, multi-agent task execution at scale.

## v1.6 Requirements

### Autonomy (AUTO)

- [ ] **AUTO-01**: L3 agents can self-direct task breakdown and planning without L1 intervention
- [ ] **AUTO-02**: Agents self-escalate to L2/L1 based on confidence thresholds
- [ ] **AUTO-03**: Context-aware tool selection based on task intent and historical outcomes
- [ ] **AUTO-04**: Progress self-monitoring with mid-task course correction
- [ ] **AUTO-05**: Autonomous handoff to L2 upon task completion or block
