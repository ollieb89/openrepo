# Plan 60-02: Escalation Notifications & Real-time Alerts

**Objective**: Implement real-time escalation alerts and escalation context panel for operators (DSH-AUTO-02).

**Context**: When autonomy tasks escalate, operators need immediate notification and full context to intervene effectively.

## Tasks

### Task 1: Escalation Alert Banner

**What**: Create global alert banner that appears when any task escalates.

**Implementation**:
```tsx
// dashboard/components/EscalationAlertBanner.tsx
import { Alert, AlertTitle, AlertDescription } from "@/components/ui/alert";
import { useEffect, useState } from "react";

export function EscalationAlertBanner() {
  const [escalations, setEscalations] = useState<EscalationEvent[]>([]);
  
  useEffect(() => {
    const unsubscribe = eventBus.subscribe(
      EVENT_ESCALATION_TRIGGERED,
      (event: AutonomyEscalationTriggered) => {
        setEscalations(prev => [...prev, event]);
      }
    );
    return unsubscribe;
  }, []);
  
  if (escalations.length === 0) return null;
  
  return (
    <div className="fixed top-4 right-4 z-50 space-y-2 max-w-md">
      {escalations.map(esc => (
        <Alert key={esc.task_id} variant="destructive">
          <ShieldAlert className="h-4 w-4" />
          <AlertTitle>Task Escalated: {esc.task_id}</AlertTitle>
          <AlertDescription>
            <p className="font-medium">{esc.reason}</p>
            <p className="text-sm mt-1">
              Confidence: {Math.round(esc.confidence * 100)}%
            </p>
            <div className="flex gap-2 mt-2">
              <Button size="sm" onClick={() => viewTask(esc.task_id)}>
                View Task
              </Button>
              <Button size="sm" variant="outline" onClick={() => dismiss(esc.task_id)}>
                Dismiss
              </Button>
            </div>
          </AlertDescription>
        </Alert>
      ))}
    </div>
  );
}
```

**Exit Criteria**:
- Banner appears immediately on escalation event
- Shows task ID, reason, confidence score
- "View Task" button navigates to task detail
- Multiple escalations stack vertically

### Task 2: Escalation Context Panel

**What**: Detailed panel showing full escalation context for operators.

**Implementation**:
```tsx
// dashboard/components/EscalationContextPanel.tsx
export function EscalationContextPanel({ taskId }: { taskId: string }) {
  const { task, events } = useTaskWithEvents(taskId);
  
  if (!task?.autonomy?.escalation) return null;
  
  const escalation = task.autonomy.escalation;
  const history = events.filter(e => 
    e.type.startsWith("autonomy.")
  );
  
  return (
    <Card className="border-red-200 bg-red-50/50">
      <CardHeader className="pb-3">
        <CardTitle className="flex items-center gap-2 text-red-700">
          <ShieldAlert className="h-5 w-5" />
          Escalation Details
        </CardTitle>
      </CardHeader>
      <CardContent className="space-y-4">
        <div className="grid grid-cols-2 gap-4">
          <div>
            <Label className="text-muted-foreground">Reason</Label>
            <p className="font-medium">{escalation.reason}</p>
          </div>
          <div>
            <Label className="text-muted-foreground">Confidence at Escalation</Label>
            <p className="font-medium text-red-600">
              {Math.round(escalation.confidence * 100)}%
            </p>
          </div>
          <div>
            <Label className="text-muted-foreground">Escalated At</Label>
            <p className="font-medium">
              {formatTimestamp(escalation.timestamp)}
            </p>
          </div>
          <div>
            <Label className="text-muted-foreground">Status</Label>
            <p className="font-medium">
              {task.autonomy.state === "escalating" ? "Awaiting Intervention" : "Resolved"}
            </p>
          </div>
        </div>
        
        <Separator />
        
        <div>
          <Label className="text-muted-foreground">Autonomy Event History</Label>
          <ScrollArea className="h-48 mt-2">
            <div className="space-y-2">
              {history.map((event, i) => (
                <div key={i} className="flex gap-2 text-sm">
                  <span className="text-muted-foreground">
                    {formatTime(event.timestamp)}
                  </span>
                  <Badge variant="outline" className="text-xs">
                    {event.type.replace("autonomy.", "")}
                  </Badge>
                  {event.payload.score && (
                    <span>confidence: {Math.round(event.payload.score * 100)}%</span>
                  )}
                </div>
              ))}
            </div>
          </ScrollArea>
        </div>
        
        {task.autonomy.state === "escalating" && (
          <div className="flex gap-2">
            <Button onClick={() => resumeTask(taskId)}>
              <Play className="h-4 w-4 mr-1" />
              Resume Task
            </Button>
            <Button variant="destructive" onClick={() => failTask(taskId)}>
              <XCircle className="h-4 w-4 mr-1" />
              Mark Failed
            </Button>
          </div>
        )}
      </CardContent>
    </Card>
  );
}
```

**Exit Criteria**:
- Shows escalation reason, confidence, timestamp
- Displays full autonomy event history
- Resume/Fail buttons for operator intervention
- Styled to indicate urgency (red accents)

### Task 3: Course Correction History

**What**: Display history of course corrections for a task.

**Implementation**:
```tsx
// dashboard/components/CourseCorrectionHistory.tsx
export function CourseCorrectionHistory({ taskId }: { taskId: string }) {
  const corrections = useAutonomyEvents(taskId, "autonomy.course_correction");
  
  if (corrections.length === 0) return null;
  
  return (
    <Card>
      <CardHeader>
        <CardTitle className="flex items-center gap-2">
          <RefreshCw className="h-4 w-4" />
          Course Corrections ({corrections.length})
        </CardTitle>
      </CardHeader>
      <CardContent>
        <Accordion type="single" collapsible>
          {corrections.map((corr, i) => (
            <AccordionItem key={i} value={`corr-${i}`}>
              <AccordionTrigger>
                <div className="flex items-center gap-2">
                  <span className="text-sm text-muted-foreground">
                    {formatTime(corr.timestamp)}
                  </span>
                  <span className="text-sm">
                    Step "{corr.failed_step.action}" failed
                  </span>
                  <Badge variant="secondary" className="ml-2">
                    {corr.recovery_steps.length} recovery steps
                  </Badge>
                </div>
              </AccordionTrigger>
              <AccordionContent>
                <div className="space-y-2 pl-4">
                  <p className="text-sm text-muted-foreground">
                    Failed action: {corr.failed_step.action}
                  </p>
                  <p className="text-sm font-medium">Recovery plan:</p>
                  <ol className="list-decimal list-inside text-sm space-y-1">
                    {corr.recovery_steps.map((step, j) => (
                      <li key={j}>{step.action}</li>
                    ))}
                  </ol>
                </div>
              </AccordionContent>
            </AccordionItem>
          ))}
        </Accordion>
      </CardContent>
    </Card>
  );
}
```

**Exit Criteria**:
- Shows count of course corrections
- Expandable items showing failed step and recovery plan
- Timestamps for each correction

### Task 4: Global Escalations Dashboard

**What**: Summary view of all escalated tasks across the system.

**Implementation**:
```tsx
// dashboard/pages/EscalationsPage.tsx
export function EscalationsPage() {
  const { data: tasks } = useTasks({ state: "escalating" });
  
  return (
    <div className="space-y-6">
      <div className="flex justify-between items-center">
        <h1 className="text-2xl font-bold">Escalated Tasks</h1>
        <Badge variant="destructive">{tasks.length} requiring attention</Badge>
      </div>
      
      {tasks.length === 0 ? (
        <Card>
          <CardContent className="py-8 text-center text-muted-foreground">
            <CheckCircle className="h-12 w-12 mx-auto mb-4 text-green-500" />
            <p>No escalated tasks. All autonomy agents running smoothly.</p>
          </CardContent>
        </Card>
      ) : (
        <div className="grid gap-4">
          {tasks.map(task => (
            <EscalationCard key={task.id} task={task} />
          ))}
        </div>
      )}
    </div>
  );
}

function EscalationCard({ task }: { task: Task }) {
  return (
    <Card className="border-red-200">
      <CardHeader className="pb-2">
        <div className="flex justify-between items-start">
          <div>
            <CardTitle className="text-base">{task.title}</CardTitle>
            <CardDescription>{task.id}</CardDescription>
          </div>
          <AutonomyStateBadge state={task.autonomy.state} />
        </div>
      </CardHeader>
      <CardContent className="space-y-2">
        <p className="text-sm">{task.autonomy.escalation.reason}</p>
        <div className="flex justify-between items-center text-sm">
          <span className="text-muted-foreground">
            Confidence: {Math.round(task.autonomy.escalation.confidence * 100)}%
          </span>
          <span className="text-muted-foreground">
            {formatRelativeTime(task.autonomy.escalation.timestamp)}
          </span>
        </div>
        <div className="flex gap-2 pt-2">
          <Button size="sm" asChild>
            <Link to={`/tasks/${task.id}`}>View Details</Link>
          </Button>
          <Button size="sm" variant="outline" onClick={() => resumeTask(task.id)}>
            Resume
          </Button>
        </div>
      </CardContent>
    </Card>
  );
}
```

**Exit Criteria**:
- Lists all escalated tasks
- Shows count badge
- Sorts by escalation time (newest first)
- Quick resume action from list view
- Link to full task detail

### Task 5: Notification Settings

**What**: Allow operators to configure notification preferences.

**Implementation**:
```tsx
// dashboard/components/NotificationSettings.tsx
export function NotificationSettings() {
  const [settings, setSettings] = useNotificationSettings();
  
  return (
    <Card>
      <CardHeader>
        <CardTitle>Escalation Notifications</CardTitle>
      </CardHeader>
      <CardContent className="space-y-4">
        <div className="flex items-center justify-between">
          <div className="space-y-0.5">
            <Label>Desktop Notifications</Label>
            <p className="text-sm text-muted-foreground">
              Show browser notification on escalation
            </p>
          </div>
          <Switch 
            checked={settings.desktop} 
            onCheckedChange={v => setSettings(s => ({ ...s, desktop: v }))}
          />
        </div>
        
        <div className="flex items-center justify-between">
          <div className="space-y-0.5">
            <Label>Sound Alerts</Label>
            <p className="text-sm text-muted-foreground">
              Play sound when escalation occurs
            </p>
          </div>
          <Switch 
            checked={settings.sound} 
            onCheckedChange={v => setSettings(s => ({ ...s, sound: v }))}
          />
        </div>
        
        <div className="space-y-2">
          <Label>Minimum Confidence Threshold</Label>
          <p className="text-sm text-muted-foreground">
            Only notify when confidence below this threshold
          </p>
          <Slider 
            value={[settings.minConfidence * 100]} 
            min={0} 
            max={100} 
            step={10}
            onValueChange={v => setSettings(s => ({ ...s, minConfidence: v[0] / 100 }))}
          />
          <p className="text-sm">{Math.round(settings.minConfidence * 100)}%</p>
        </div>
      </CardContent>
    </Card>
  );
}
```

**Exit Criteria**:
- Desktop notification toggle
- Sound alert toggle
- Configurable confidence threshold
- Settings persisted to localStorage
