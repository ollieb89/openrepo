Create a new file `orchestration/migrate_state.py`. This is a standalone script run via `python3 orchestration/migrate_state.py [--project pumplai]`.

**Implementation per RESEARCH.md Pattern 5 and locked decisions:**

1. **Imports**: `argparse`, `json`, `shutil`, `sys`, `pathlib.Path`. Import `JarvisState` from `orchestration.state_engine` and `get_state_path`, `_find_project_root` from `orchestration.project_config`. Add `sys.path.insert(0, str(Path(__file__).parent.parent))` before orchestration imports so it works when run directly.

2. **CLI**: `argparse` with `--project` flag defaulting to `"pumplai"`.

3. **Paths**:
   - `old_path = project_root / "workspace" / ".openclaw" / "workspace-state.json"` (hardcoded — this is the legacy location)
   - `new_path = get_state_path(args.project)` (from project_config API)

4. **Idempotency check**: If `old_path` does not exist:
   - If `new_path` exists: print "Already migrated" and return 0
   - If neither exists: create empty state at new_path via `JarvisState(new_path)` and return 0

5. **In-flight guard**: Read state via `JarvisState(old_path).read_state()`. Define `IN_FLIGHT_STATUSES = {'spawned', 'running', 'in_progress', 'starting', 'testing'}`. Check all tasks — if any have a status in IN_FLIGHT_STATUSES, print "ERROR: Cannot migrate while tasks are in-flight", list blocking task IDs and statuses, and return 1. Do NOT include `"pending"` — pending tasks have no live container.

6. **Before/after summary**: Print old path, new path, task count, and each task ID with status.

7. **Backup**: Create `workspace/.openclaw/.backup/` directory. Copy old state to `workspace/.openclaw/.backup/workspace-state.json.bak` using `shutil.copy2` (preserves metadata).

8. **Copy to new location**: `new_path.parent.mkdir(parents=True, exist_ok=True)`, then `shutil.copy2(old_path, new_path)`.

9. **Verify copy integrity**: Load new_path as JSON, assert task count matches original.

10. **Sentinel old path**: Write a JSON sentinel to old_path:
    ```python
    {
        "migrated": True,
        "new_path": str(new_path),
        "error": f"This state file has been migrated to {new_path}. Update callers to use project_config.get_state_path()."
    }
    ```

11. **Also migrate snapshots**: Check if old snapshot dir `workspace/.openclaw/snapshots/` exists and has contents. If so:
    - Copy entire directory tree to `get_snapshot_dir(args.project)` using `shutil.copytree` with `dirs_exist_ok=True`
    - Print snapshot migration summary

12. Print "Migration complete." and return 0.

Add `if __name__ == "__main__": sys.exit(main())` at the bottom.

**Anti-patterns to avoid:**
- Do NOT use `shutil.move` — always copy first, then sentinel
- Do NOT auto-resolve stale tasks — operator decides
- Do NOT add a `--force` flag — per locked decision
