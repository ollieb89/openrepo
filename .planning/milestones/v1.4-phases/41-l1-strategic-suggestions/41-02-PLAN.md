**Constants:**
```typescript
const OPENCLAW_ROOT = process.env.OPENCLAW_ROOT || '/home/ollie/.openclaw';
const MAX_DIFF_LINES = 100;
const FORBIDDEN_PATTERNS: RegExp[] = [
  /cap_drop/i,
  /no-new-privileges/i,
  /LOCK_TIMEOUT/i,
  /shell=/i,
  /exec\s*\(/i,
  /subprocess/i,
  /os\.system/i,
  /`[^`]+`/,        // backtick shell commands
  /\$\([^)]+\)/,    // shell substitution $(...)
];
```

**`validateDiffText(diffText: unknown): { valid: boolean; reason?: string }`**
- If `diffText === null || diffText === undefined || diffText === ''` → `{ valid: false, reason: 'diff_text is required and must be a non-empty string' }` (Pitfall 4 from RESEARCH.md)
- If `typeof diffText !== 'string'` → `{ valid: false, reason: 'diff_text must be a string' }`
- `const lines = diffText.split('\n')` — if `lines.length > MAX_DIFF_LINES` → `{ valid: false, reason: \`Diff exceeds ${MAX_DIFF_LINES} lines (got ${lines.length})\` }`
- For each pattern in `FORBIDDEN_PATTERNS`: if `pattern.test(diffText)` → `{ valid: false, reason: \`Diff contains forbidden pattern: ${pattern}\` }`
- Return `{ valid: true }`

**Helper functions:**
```typescript
function suggestionsPath(projectId: string): string {
  return path.join(OPENCLAW_ROOT, 'workspace', '.openclaw', projectId, 'soul-suggestions.json');
}
function soulOverridePath(projectId: string): string {
  return path.join(OPENCLAW_ROOT, 'projects', projectId, 'soul-override.md');
}
async function rerenderSoul(projectId: string): Promise<void> {
  const execFileAsync = promisify(execFile);
  await execFileAsync('python3', [
    path.join(OPENCLAW_ROOT, 'orchestration', 'soul_renderer.py'),
    '--project', projectId, '--write', '--force',
  ], { cwd: OPENCLAW_ROOT });
}
```

**`POST` handler** — body: `{ action: 'accept' | 'reject', project: string, diff_text?: string, rejection_reason?: string }`:
1. Parse body — return 400 if `action` or `project` missing
2. Read `soul-suggestions.json` — return 404 if suggestion id not found
3. **If `action === 'accept'`:**
   - Call `validateDiffText(body.diff_text)` — if not valid: return 422 with `{ error: reason }`
   - `await fs.mkdir(path.dirname(soulOverridePath(project)), { recursive: true })` — creates dir if missing (Pitfall 5 from RESEARCH.md)
   - `await fs.appendFile(soulOverridePath(project), '\n\n' + diff_text.trim() + '\n', 'utf-8')` — appends to soul-override.md
   - `await rerenderSoul(project)` — re-renders SOUL.md via soul_renderer.py
   - Update suggestion in JSON: `status = 'accepted'`, `accepted_at = Date.now() / 1000`
   - Write updated JSON back to `suggestionsPath(project)`
   - Return `{ ok: true, message: 'Applied to soul-override.md' }`
4. **If `action === 'reject'`:**
   - Update suggestion in JSON: `status = 'rejected'`, `rejected_at = Date.now() / 1000`, `rejection_reason = body.rejection_reason || null`, `suppressed_until_count = suggestion.evidence_count * 2`
   - Write updated JSON back to `suggestionsPath(project)`
   - Return `{ ok: true }` — rejection reason memorization deferred to L2 (CLI does it, dashboard doesn't need to call memU directly)
5. Return 400 for unknown action
