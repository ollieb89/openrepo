### Task 3: Tune suggest.py thresholds for small datasets

**Files:**
- Modify: `packages/orchestration/src/openclaw/cli/suggest.py:30`

**Current:**
```python
MIN_CLUSTER_SIZE = 3           # ADV-01 requirement: ≥3 similar rejections
```

**Analysis:**
- Current threshold requires ≥3 similar rejections to form a cluster
- On small datasets (<10 memories), this often produces 0 suggestions
- Need adaptive threshold based on dataset size

**Change:**
Replace constant with dynamic function that scales based on dataset size:

```python
MIN_CLUSTER_SIZE = 3           # ADV-01 requirement: ≥3 similar rejections
MIN_CLUSTER_SIZE_SMALL = 2     # Relaxed threshold for small datasets (<10 items)
SMALL_DATASET_THRESHOLD = 10   # Memories below this count use relaxed threshold

def _get_min_cluster_size(memory_count: int) -> int:
    """
    Return adaptive min cluster size based on dataset scale.
    
    Small datasets (<10 memories) use relaxed threshold (2)
    to avoid filtering out all suggestions.
    Larger datasets use strict threshold (3) per ADV-01.
    """
    if memory_count < SMALL_DATASET_THRESHOLD:
        return MIN_CLUSTER_SIZE_SMALL
    return MIN_CLUSTER_SIZE
```

Then update the cluster formation logic to use this function instead of the constant.

**Verify:**
- Run: `uv run pytest packages/orchestration/tests/test_suggest.py -v`
- Expected: All tests pass
- Create test with 4 memories: suggestions should be generated (was 0 before)
