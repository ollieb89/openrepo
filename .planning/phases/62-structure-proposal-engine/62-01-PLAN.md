  <behavior>
    - save_topology(project_id, graph) writes to workspace/.openclaw/{project_id}/topology/current.json with LOCK_EX
    - load_topology(project_id) reads current.json with LOCK_SH, returns TopologyGraph or None if not found
    - append_changelog(project_id, entry) appends to topology/changelog.json with LOCK_EX
    - load_changelog(project_id) reads changelog.json with LOCK_SH, returns list of dicts
    - Creates topology/ directory if it does not exist
    - Uses .tmp + rename for atomic writes (crash-safe)
    - Creates .bak before overwriting current.json
    - Topology files are NEVER at workspace-state.json path (separate directory)
  </behavior>
  <action>
Create `packages/orchestration/src/openclaw/topology/storage.py`:

1. `_topology_dir(project_id: str) -> Path` — returns `get_project_root() / "workspace" / ".openclaw" / project_id / "topology"`, creates dir if missing.

2. `save_topology(project_id: str, graph: TopologyGraph) -> None`:
   - Get path = `_topology_dir(project_id) / "current.json"`
   - If path exists, copy to `current.json.bak`
   - Write `graph.to_json()` to `current.json.tmp`
   - Acquire LOCK_EX on `.tmp` file
   - Rename `.tmp` → `current.json` (atomic)
   - Follow state_engine.py fcntl.flock pattern

3. `load_topology(project_id: str) -> Optional[TopologyGraph]`:
   - Path = `_topology_dir(project_id) / "current.json"`
   - If not exists, return None
   - Open with LOCK_SH, read JSON, return `TopologyGraph.from_json()`
   - On corruption, try `.bak` recovery (log warning)

4. `append_changelog(project_id: str, entry: dict) -> None`:
   - Path = `_topology_dir(project_id) / "changelog.json"`
   - Read existing entries (or empty list), append entry, write back
   - LOCK_EX for read-modify-write

5. `load_changelog(project_id: str) -> list[dict]`:
   - Read with LOCK_SH, return list of dicts (empty list if missing)

Add tests to `test_topology_models.py` (or new file if cleaner):

- test_topology_dir_created_automatically
