  <name>Task 1: Create MemoryClient module with enforced scoping</name>
  <files>orchestration/memory_client.py</files>
  <action>
Create `orchestration/memory_client.py` with the following components:

**AgentType enum** (str, Enum):
- Values: `l2_pm`, `l3_code`, `l3_test`
- str subclass so it serializes directly in JSON payloads without `.value`

**MemorizeResult dataclass** (frozen=True):
- `accepted: bool`
- `message: str`

**RetrieveResult dataclass** (frozen=True):
- `items: List[dict]` (default_factory=list)

**MemoryClient class:**
- Constructor: `__init__(self, base_url: str, project_id: str, agent_type: AgentType)` — all three positional, no defaults. Python raises TypeError if any omitted. Store `base_url.rstrip("/")`, `project_id`, `agent_type`. Set `self._client: Optional[httpx.AsyncClient] = None`.
- Async context manager: `__aenter__` creates `httpx.AsyncClient(base_url=self.base_url)`, `__aexit__` calls `await self._client.aclose()`.
- Module-level timeouts: `TIMEOUT_RETRIEVE = httpx.Timeout(3.0, connect=2.0)`, `TIMEOUT_MEMORIZE = httpx.Timeout(10.0, connect=2.0)`.
- Module-level logger: `logger = get_logger("memory_client")` — import from `orchestration.logging`.

**health() method** → bool:
- GET `/health`, timeout 3s
- Return `True` if 200 and response JSON has `"memu_initialized": True`
- Catch `httpx.TimeoutException`, `httpx.ConnectError`, `httpx.HTTPStatusError` → log warning with project_id, return `False`

**memorize(content: str, category: str = "general") method** → Optional[MemorizeResult]:
- POST `/memorize` with JSON payload:
  ```json
  {
    "resource_url": content,
    "modality": "conversation",
    "user": {
      "user_id": self.project_id,
      "agent_type": self.agent_type.value
    }
  }
  ```
- Timeout: TIMEOUT_MEMORIZE (10s)
- On 202: return `MemorizeResult(accepted=True, message="accepted")`
- Catch exceptions → log warning with project_id + agent_type + error, return `None`

**retrieve(query: str) method** → List[dict]:
- POST `/retrieve` with JSON payload:
  ```json
  {
    "queries": [{"role": "user", "content": query}],
    "where": {"user_id": self.project_id}
  }
  ```
  The `where` clause maps `project_id → user_id` which is how memU filters. This is the core of SCOPE-01.
- Timeout: TIMEOUT_RETRIEVE (3s)
- On 200: parse response. The Phase 26 `/retrieve` endpoint returns `memu.retrieve()` result directly — the shape is a list. If response is a list, return it directly. If response is a dict with an `items` key, return `data["items"]`. Otherwise return `[]`.
- Catch exceptions → log warning with project_id + error, return `[]`

**Module-level helper:**
```python
MEMU_SERVICE_URL = os.environ.get("MEMU_SERVICE_URL", "http://memu-server:18791")
```

Import `os` at top level. This constant is for downstream callers (spawn.py, etc.) to use when constructing MemoryClient.
