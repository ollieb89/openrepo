# Plan 60-01: Autonomy State Dashboard Components

**Objective**: Implement dashboard UI components for displaying autonomy state, confidence scores, and selected tools per task (DSH-AUTO-01).

**Context**: Phase 60 addresses the DSH-AUTO-01 requirement gap. The autonomy framework emits rich events but operators cannot see task autonomy state in the dashboard.

### Task 2: Confidence Score Visualization

**What**: Create confidence score display with progress bar and numeric value.

**Implementation**:
```tsx
// dashboard/components/ConfidenceIndicator.tsx
import { Progress } from "@/components/ui/progress";

export function ConfidenceIndicator({ score, threshold = 0.4 }: { 
  score: number;
  threshold?: number;
}) {
  const percentage = Math.round(score * 100);
  const isLow = score < threshold;
  
  return (
    <div className="flex items-center gap-2">
      <Progress 
        value={percentage} 
        className={isLow ? "bg-red-200" : "bg-green-200"}
      />
      <span className={isLow ? "text-red-600" : "text-green-600"}>
        {percentage}%
      </span>
      {isLow && <AlertTriangle className="h-4 w-4 text-red-500" />}
    </div>
  );
}
```

**Exit Criteria**:
- Progress bar shows score percentage
- Color changes to red when below threshold
- Numeric percentage displayed
- Tooltip shows threshold value

### Task 3: Selected Tools Display

**What**: Show which tools an L3 agent has selected for the task.

**Implementation**:
```tsx
// dashboard/components/SelectedTools.tsx
const toolIcons = {
  file_read: "FileText",
  file_write: "FileEdit",
  shell_execution: "Terminal",
  web_search: "Search",
  git_operations: "GitBranch",
};

export function SelectedTools({ tools }: { tools: string[] }) {
  if (tools.includes("all")) {
    return <Badge variant="secondary">All Tools</Badge>;
  }
  
  return (
    <div className="flex gap-1 flex-wrap">
      {tools.map(tool => (
        <Badge key={tool} variant="outline" className="gap-1">
          <Icon name={toolIcons[tool]} className="h-3 w-3" />
          {tool.replace("_", " ")}
        </Badge>
      ))}
    </div>
  );
}
```

**Exit Criteria**:
- Shows all 5 tool categories with icons
- Displays "All Tools" when no restrictions
- Responsive layout for many tools

### Task 5: WebSocket Event Listener

**What**: Connect to event bus WebSocket to receive real-time autonomy updates.

**Implementation**:
```tsx
// dashboard/hooks/useAutonomyEvents.ts
export function useAutonomyEvents(taskId: string) {
  const [state, setState] = useState<AutonomyState | null>(null);
  
  useEffect(() => {
    const ws = new WebSocket(`ws://localhost:8080/events?task=${taskId}`);
    
    ws.onmessage = (event) => {
      const data = JSON.parse(event.data);
      if (data.type === "autonomy.state_changed") {
        setState(data.payload.state);
      }
    };
    
    return () => ws.close();
  }, [taskId]);
  
  return state;
}
```

**Exit Criteria**:
- WebSocket connects on component mount
- State updates in real-time from events
- Handles reconnection on disconnect
- Cleanup on unmount
