   - Lowercase the title.
   - For each area in config.area_keywords, check if any keyword appears in the title.
   - Return `(area_name, inferred=True)` for first match.
   - If no match → return `("Admin", True)` as fallback (per CONTEXT.md: best guess + tag as uncertain).

   - Sort keys alphabetically, join as `"key:value"` separated by `"|"`.

```python
def _parse_batch(capture: dict) -> list[dict]:
    title = capture.get("title", "")
    # Check for batch separators
    if "," in title and not any(c in title for c in [". ", "? ", "! "]):
        # Likely a comma-separated list
        titles = [t.strip() for t in title.split(",") if t.strip()]
    elif "\n" in title:
        titles = [t.strip() for t in title.split("\n") if t.strip()]
    else:
        titles = [title]

    items = []
    for t in titles:
        item = dict(capture)
        item["title"] = t
        items.append(item)
    return items
```
