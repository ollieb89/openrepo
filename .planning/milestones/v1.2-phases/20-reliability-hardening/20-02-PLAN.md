  <action>
Create `orchestration/config_validator.py` with two main validation functions:

**1. `validate_project_config(config: dict, manifest_path: str) -> None`**

Validates a project.json dict. Raises `ConfigValidationError` on failure.

Required fields (per user decision): `workspace`, `tech_stack`.
- Check each required field exists and is the correct type:
  - `workspace`: must be a non-empty string
  - `tech_stack`: must be a dict
- For each missing/invalid field, collect errors (collect-all strategy for project.json — one file, show all problems at once)
- Error message format per user decision:
  ```
  project.json ({manifest_path}): missing required field "workspace". Add a workspace path pointing to your project directory.
  ```
  or for type errors:
  ```
  project.json ({manifest_path}): field "tech_stack" must be an object, got str. Define tech_stack as {"frontend": "...", "backend": "..."}.
  ```
- If errors collected, raise `ConfigValidationError` with all errors joined by newlines

**2. `validate_agent_hierarchy(config: dict, config_path: str) -> None`**

Validates the `agents.list` array in openclaw.json. Raises `ConfigValidationError` on failure.

Checks:
- Every agent's `reports_to` (if not null) must reference an existing agent `id` in the list
  - Error: `openclaw.json ({config_path}): agent "pumplai_pm" reports_to "nonexistent_agent" which does not exist. Check the agent ID spelling or add the missing agent.`
- Level constraint: an agent's level must be greater than the level of its `reports_to` target (L3 reports to L2, L2 reports to L1 — not L3 to L3 or L2 to L1)
  - Error: `openclaw.json ({config_path}): agent "worker_a" (level 3) reports_to "worker_b" (level 3). An agent must report to a higher-tier agent (lower level number).`
- L1 agents (level 1) must have `reports_to: null`
  - Error: `openclaw.json ({config_path}): agent "main" is level 1 but reports_to "other". Level 1 agents must have reports_to: null.`
- Collect all errors and raise with all joined by newlines (fail-fast per file, collect-all within a file)

**3. `ConfigValidationError(Exception)` class**
- Custom exception with `errors: list[str]` attribute
- `__str__` returns all errors joined by `\n`

**4. Import `get_logger`** and log each validation call at DEBUG ("Validating project config for {manifest_path}") and errors at ERROR level.

def load_and_validate_openclaw_config() -> Dict[str, Any]:
    """Load openclaw.json and validate agent hierarchy.

    Raises:
        ConfigValidationError: If agent hierarchy has invalid reports_to or level constraints
    """
    root = _find_project_root()
    config_path = root / "openclaw.json"
    with open(config_path) as f:
        config = json.load(f)
    validate_agent_hierarchy(config, str(config_path))
    return config
