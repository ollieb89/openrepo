<context>
**Phase:** 0 - Shared Infrastructure
**Wave:** 0.5
**Goal:** Observability, alerting, and GDPR/EU AI Act foundations. Non-negotiable for enterprise sales.
**Context:** Global shared system data applies to Prompt Registries and AI Act categories, while Audit Trails and GDPR consent are strictly org-scoped.
</context>

<requirements>
- Prometheus metrics exporter middleware.
- Grafana dashboard provisioning (extract logic from `scaling_monitoring_tools.py`).

- TTL policies for conversation logs (PG cron or async task).
- Immutable append-only audit trail logging.
- EU AI Act risk categorization registry.
- Git-versioned prompt registry.
- Gate G0: Integration test suite proving end-to-end request lifecycle from auth -> billing -> LLM routing -> audit log.
</requirements>
