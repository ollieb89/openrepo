    - Design a Chain-of-Thought (CoT) prompt that instructs Phi-3 to:

      4. Find the "Smoking Gun" citation (exact quote).
    - Use Regex to hoist Linear IDs (`[A-Z0-9]{2,}-\d+`) and Slack Mentions (`<@U[A-Z0-9]+>`) from the LLM output into the `Decision` metadata fields.
    - Implement a validation step to ensure the LLM-provided citation actually exists in the original thread text (to prevent hallucinations).
