# Plan: Phase 0, Wave 0.4 - LLM Abstraction Layer

<context>
**Phase:** 0 - Shared Infrastructure
**Wave:** 0.4
**Goal:** SmartRouter for model selection, cost tracking, and fallback chains. Margin-critical component.
**Context:** 
- Provider Defaults: Flash-Lite/DeepSeek V3.2 (simple), Flash 2.5/Haiku/Sonnet/DeepSeek R1 (standard), Opus/GPT-5.2 (complex).
- Resilience: Strict Circuit Breaker (opens after 5 failures/60s).
- Cost: Granular Token Tracking meticulously matching provider token prices, aggregated per org.
</context>

<requirements>
- `SmartRouter` base provider interface abstraction.
- Provider adapters for Anthropic, OpenAI, Google (async, streaming).
- Config-driven routing rules mapping task types to the cheapest capable model tier.
- Per-request cost tracking stored in PostgreSQL, attributed to `org_id`.
- Fallback chain + Strict Circuit Breaker.
- Gate G0.4: Mock routing test shows correct model selection, fallback triggers, and cost attribution.
</requirements>
