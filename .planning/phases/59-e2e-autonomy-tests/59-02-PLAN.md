# Plan 59-02: Retry & Escalation Path Tests

**Objective**: Implement E2E tests for retry path (BLOCKED → recovery → COMPLETE) and escalation path (BLOCKED → ESCALATING → pause → resume).

**Context**: Building on 59-01 infrastructure, test the failure recovery and escalation scenarios that are core to the autonomy framework.

---

## Tasks

**Implementation**:
```python

    # 2. Wait for BLOCKED state
```

```json
// responses/fail_then_succeed.json
```

**Exit Criteria**:
- Test verifies BLOCKED state reached
- Test verifies AutonomyCourseCorrection event
- Test verifies recovery step execution
- Test passes with correct state transitions

**Implementation**:

```python
    """Verify escalation triggers pause and can be resumed."""
    # 1. Spawn task with low confidence threshold (0.1)
    # 2. Mock LLM to fail repeatedly
    # 3. Wait for ESCALATING state
    # 4. Verify AutonomyEscalationTriggered event
```

**Exit Criteria**:
- Test triggers escalation via confidence threshold
- Test verifies escalation event emitted
- Test verifies pause state in sentinel file
- Test verifies resume functionality
