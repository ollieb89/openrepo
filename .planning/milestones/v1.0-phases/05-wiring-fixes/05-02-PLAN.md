<task type="auto">
  <name>Task 1: Create startup initialization module</name>
  <files>orchestration/init.py</files>
  <action>
Create `orchestration/init.py` — a startup initialization module that guarantees all required directories exist.

**NOTE:** This file is `orchestration/init.py`, NOT `orchestration/__init__.py` (which already exists as the package init). Name it `init.py` to avoid confusion — it is a startup initialization module, not a Python package initializer.

**Functions to implement:**

1. `initialize_workspace(project_root: Path = None) -> Dict[str, Any]`:
   - Accept optional project_root (default: auto-detect from script location, walking up to find openclaw.json)
   - Create required directories idempotently using `Path.mkdir(parents=True, exist_ok=True)`:
     - `{project_root}/workspace/.openclaw/snapshots/`
   - Verify the directory was created or already exists
   - Return a dict: `{"snapshots_dir": str, "created": bool, "already_existed": bool}`
   - Log what happened (created vs already existed) to stdout

2. `verify_workspace(project_root: Path = None) -> Dict[str, bool]`:
   - Check that all required workspace directories exist
   - Check that orchestration modules are importable
   - Return dict of checks: `{"snapshots_dir": True/False, "state_file_dir": True/False, "orchestration_importable": True/False}`

3. `main()`:
   - CLI entrypoint: `python3 orchestration/init.py`
   - Calls `initialize_workspace()` then `verify_workspace()`
   - Prints results with ANSI colors (green check / red X)
   - Exit 0 if all checks pass, exit 1 if any fail

**Key design decisions (per user locks):**
- Idempotent: uses `exist_ok=True`, checks existence first, safe to run multiple times
- Leave snapshots directory empty (no baseline snapshot — first L3 task creates first snapshot)
- Document manual invocation: `python3 orchestration/init.py` (baked into startup means it runs when system starts, but for now documented as manual invocation until a startup script exists)

**Also update `orchestration/__init__.py`** to export the new module:
Add `from .init import initialize_workspace, verify_workspace` to `__init__.py` and update `__all__`.

**Do NOT use external dependencies** — only Python stdlib (pathlib, json, sys, os).
