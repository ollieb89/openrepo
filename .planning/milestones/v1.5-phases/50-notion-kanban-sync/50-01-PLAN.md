Create `packages/orchestration/src/openclaw/event_bus.py` with the following:

1. Module-level `_handlers: Dict[str, List[Callable]]` using `defaultdict(list)`.
2. `subscribe(event_type: str, handler: Callable[[Dict[str, Any]], None]) -> None` — appends handler to `_handlers[event_type]`.
3. `emit(envelope: Dict[str, Any]) -> None` — reads `event_type` from envelope, iterates over `_handlers[event_type]`, spawns each handler call in a `threading.Thread(daemon=True)`. Returns immediately (fire-and-forget).
4. `_call_handler(handler, envelope)` — wraps handler call in try/except, logs exceptions via `logging.getLogger("openclaw.event_bus")`. NEVER re-raises.
5. `clear_handlers() -> None` — for testing: resets `_handlers` to empty defaultdict.

CRITICAL constraints:
- Zero imports from `openclaw` package at module level (avoids circular imports per Research pitfall 3).
- Only stdlib imports at module level: `threading`, `logging`, `collections.defaultdict`, `typing`.
- Logger uses `logging.getLogger("openclaw.event_bus")` — consistent with codebase pattern.

```python
    from .event_bus import emit
    from datetime import datetime, timezone
    _status_to_event = {
        "in_progress": "phase_started",
        "completed": "phase_completed",
        "waiting": "phase_blocked",
    }
    evt_type = _status_to_event.get(status)
    if evt_type and task_id:
        emit({
            "event_type": evt_type,
            "occurred_at": datetime.now(timezone.utc).isoformat(),
            "project_id": get_active_project_id(),
            "phase_id": task_id.split("-")[0] if "-" in task_id else task_id,
            "container_id": None,
            "payload": {"task_id": task_id, "status": status, "activity_entry": activity_entry},
        })
except Exception:
    logger.debug("Event emission failed (non-blocking)", exc_info=True)
```
