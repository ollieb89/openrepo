6. **Add `_detect_drift(spec: AgentSpec)`** method:
   ```python
   def _detect_drift(self, spec: AgentSpec) -> None:
       """Warn when per-agent config.json disagrees with openclaw.json on identity fields."""
       central = self._openclaw_json_data.get(spec.id)
       if central is None:
           return  # filesystem-only agent — no drift possible, already warned as "new"
       per_agent_vals = {
           "name": spec.name,
           "level": int(spec.level),
           "reports_to": spec.reports_to,
       }
       for field in _DRIFT_IDENTITY_FIELDS:
           if central.get(field) != per_agent_vals.get(field):
               _logger.warning(
                   "Agent '%s' drift: field '%s' is '%s' in openclaw.json but '%s' "
                   "in agents/%s/agent/config.json. Run `openclaw agent sync` to update.",
                   spec.id, field, central.get(field), per_agent_vals.get(field), spec.id,
               )
   ```

7. **Add `_detect_orphans()`** method — call at end of `_load()`. For each agent in `self._openclaw_json_data` that was never updated to source="both" or "agents_dir", it's an orphan:

   def _detect_orphans(self) -> None:
       for aid, spec in self._agents.items():

                   "agents/%s/agent/config.json. Run `openclaw agent init %s` to scaffold.",
